+ Html hypertest markup language

- hyperText
	하이퍼링크가 적용된 문서(텍스트)
	다른 문서와 링크로 연결된 문서(웹문서)

- markup language
	태그(tag)를 사용하여 문서 또는 데이터의 구조를 표현하는 언어

	
 - 웹페이지의 정적인 구조를 표현하는 언어이다

 - 웹 ui 레이아웃을 구성한다.

 - 태그를 이용하여 개발한다.


+ html 태그, tag

 - html문서에서 사용하는 요소(문법)
 - <여는태그> 와 </닫는태그> 두 개를 한 쌍으로 구성된다.
	
	ex) <button></button> - button 태그
	    <div> </div> - div 태그

 - html문서의 기본 태그 : <html>, <head>, <body>
 - 기본 태그 3가지는 html문서에 반드시 포함되어야 한다.

+ 기본 태그들의 형식
<!DOCTYPE html>
<html>
 <head>
 </head>
 <body>

 </body>
</html>

- html 태그가 html 문서의 최상위 부모태그이다.
- html 태그가 head, body 태그를 자식태그로 가진다.
- 태그들이 계층적인 구조를 가진다.

- <html> 태그
	html문서의 최상위 태그
	문서 내 모든 태그를 감싸고 있다.

- <head> 태그
	html문서의 정보를 입력하는 태그
	html문서 자체에 대한 정보 또는 설정값들을 입력한다	
	

- <body> 태그
	화면에 보여진 내용을 작성하는 곳
