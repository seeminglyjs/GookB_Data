+ PL/SQL
 - SQL에 절차적 언어 특성을 확장한 것
 - 기본 SQL에서 일반적으로 제공되지 않는 변수, 조건문, 반복문 등 사용할 수 있도록 제공하는 SQL
 -> SQL에 프로그래밍 처리가 추가된 것


 - 익명 블록, Anonymous Block
	곧바로 실행시켜 사용하는 PL/SQL

 - 저장 프로시저, Stored Procedure
	익명 블록을 이름을 붙여 저장한 PL/SQL 블록

 - 함수, Function
	매개변수(입력)과 리턴(출력)이 있는 PL/SQL 블록
	단일행함수처럼 사용할 수 있다.	

 - 트리거, Trigger
	특정 상황에 수행되도록 작성해놓은 PL/SQL 블록
	
	테이블의 데이터가 변화했을 때 수행되도록 만든다.
	(INSERT, UPDATE, DELETE 수행 시 ) 	


+ PL/SQL의 기본 구문 형식

	DECLATE
		변수 선언
	BEIGN
		수행할 코드
 	EXCEPTION
		예외 처리
	END;

	** BEGIN, END 는 필수 요소

	** DECLATE, EXCEPTION은 필수가 아님
