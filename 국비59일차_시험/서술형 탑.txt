DriverManager

자바와 sql(db)을 연결시켜주기 위한 JDBC Api 클래스이다. 자바 내부에서 Connection 객체에 class를 매개값으로 주어 연결하여 사용한다.

PreparedStatement

기존의 Statement 클래스의 기능을 상속받은 클래스로써 sql 쿼리구문에 ? 기호를 사용할 수 있게 된다 해당 ?는 sql 내부의 value값이 들어가는 자리에 위치하게 된다. 이후 해당 ? 를 setXXX(index, ?) 를 통해서 각 속성 타입에 맞게 메소드타입과 값을 대입해주게 되면, 정상적으로 쿼리문이 PreparedStatement 객체에 저장된다.
