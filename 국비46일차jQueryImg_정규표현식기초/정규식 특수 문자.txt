+ 정규식 특수 문자

 - \ (원)
	다음으로 문자가 특수 문자가 아니라 평범한 문자로 인식되록 만든다

	다음으로 문자와 결합되어 특수 문자로 사용되도록 만든다

	ex) /a*/  ->  "a" 문자가 0개 또는 여러 개인 패턴
	ex) /a\*/ ->  "a*"인 단순 문자 패턴

	ex) /w/   -> "w"인 단순문자 패턴
	ex) /\w/  -> 문자(word)를 인식하는 특수 문자 패턴

 - ^
	데이터의 시작에 매칭시킨다
	정규식을 표현하는 / /의 앞쪽 /에 붙여서 사용한다
	ex) /^A/  -> 문자열의 시작이 대문자 A인 패턴
	"an A" - 매칭되지 않음
	"A b" - 매칭 O
	"An - 매칭 O
	
 - $
	데이터의 끝에 매칭시킨다
	정규식을 표현하는 / / 의 뒤쪽 /에 붙여서 사용한다
	
	ex) /t$/  -> 문자열의 끝이 소문자 t인 패턴
	"eater" - 매칭 X 
	"eat" - 매칭 O


** ^ 와 $ 를 같이 사용할 경우
	ex) /^eat$/ -> "eat"이라는 온전한 단어에만 매칭

	-> ^ $로 감싸진 하나의 단어를 인식한다

	-> <from>검증할 때 많이 사용하는 패턴
