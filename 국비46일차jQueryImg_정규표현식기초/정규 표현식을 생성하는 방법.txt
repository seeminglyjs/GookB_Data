+ 정규 표현식을 생성하는 방법

1. 정규식 리터럴 상수
	패턴 문자열을 /(슬래스)로 감싸서 표현한다
	ex) var regEx = /pattern/

2. RegExp 내장객체로 생성
	패턴문자열을 RegExp객체의 생성자의 매개변수로 전달하며 표현한다
	ex) var regObj = new RegExp("pattern")


+ 정규표현식에서 사용하는 패턴

 1. 단순 패턴
	완전히 일치하는 문자열을 대응시킬 때 사용한다
	ex) /abc/ -> "abc" 문자열과 매칭
	ex) /hello/ -> "hello" 문자열과 매칭
 2. 특수 문자 패턴
	특수한 형식의 패턴을 적용하여 문자열을 대응시킬 때 사용
------------------------------------------------------
+ 정규 표현식에서 사용하는 플래그(옵션)
 - 플래그를 적용하지 않으면 기본패턴만 찾아낸다
 -> 처음 매칭되는 패턴문자열 1개를 찾는다

 - g , global
	패턴에 매칭된 모든 문자
	반복으로 나오는 패턴문자열을 모두 찾아낸다	

 - i, ignore case, case-insensitive
	영어 대소문자를 구분하지 않고 찾아낸다

 - m, multi line
	줄바꿈(개행)을 포함한 문자열에서도 동작하도록 설정

 - u, unicode
	유니코드 문자를 정규식 매칭에 포함시킨다
