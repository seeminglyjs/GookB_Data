

익명객체 : 단발성으로 사용하기위해 PL/ SQL을 작성된 코드이며,저장되지 않고 선언된 즉시 실행시켜 사용하는 코드 문법이다.

프로시저 : PL/SQL을 단발성이 아닌 하나의 저장객체를 만들어 언제든지 EXEC / EXECUTE를 이용해 호출하여 사용할 수 있게 하는 코드 문법이다.

FUNCTION : 자바의 메소드처럼 SQL 내부에서도 사용자 임의이 필요한 기능을 만들어 함수로 사용할 수 있게 하는 코드 문법이다.

1. 스칼라 서브쿼리 : SELECT 절에서 조건을 부여하는 서브쿼리
	

2. 서브쿼리 : WHERE 절에 조건을 부여하여 사용하는 서브쿼리
	

3. 인라인 뷰 : FROM 절에 서브쿼리 조건을 부여해서 사용하는 서브쿼리

4. 다중행 서브쿼리 : WHERE 절에 사용하는 서브쿼리로써 키워드로는 IN / ANY / ALL / EXISTS 를 사용하여 키워드 조건에 따라 비교한다.
조회 결과가 2가지 행이상이다.

5. 다중열 서브쿼리 : WHERE 절에 두 개 이상의 컬럼이 괄호로 묶어 들어가며, 이후 IN / ANY / ALL / EXISTS 키워드 조건에 따라 비교한다 조회결과가 2가지 이상의 열이다.
