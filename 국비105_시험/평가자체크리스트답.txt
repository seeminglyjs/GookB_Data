사용자가 id와 pw를 입력하고 로그인 버튼을 클릭한다.

-----------

시작조건 : 로그인 UI가 구현되어 있어야 하며, 데이터베이스에 사용자정보가 들어갈 테이블과 해당 테이블에 사용자 정보가 저장되어 있어야 한다.

종료조건 : 사용자가 입력한 id과 다른 사용자와 중복되지 않으며, DB 에 저장된 사용자의 id와 pw가 일치할 때


------------

예상결과 : 로그인에 성공하셨습니다. 메시지 출력
실제결과 : 예상과 같이 로그인에 성공하셨습니다. 메시지 출력



------------

사용자가 입력한 데이터를 곧바로 DB에 전달하여 일치여부를 판단하는 것보다 DB 조회전 1차적으로 정규식을 이용해 사용자가 DB 정해진 데이터 타입과 규격에 맞게 입력했는지 확인해 본다.
