+ 테이블 태그
 - 표를 표현하는 태그

- <table>
- <tr>, <th>, <td>

-<caption>, <thread>, <tbody>, <tfoot>

- table : 표를 만들 때 사용하는 태그, 자식 태그들을 이용하여 행
과 열을 표현한다.

 - tr : table row, 테이블의 각 행을 표현한다.
	<table> 태그의 자식요소로 사용된다.

 - th : table head, 테이블의 제목을 나타내는 셀을 표현한다.

 - td : table data, 테이블의 데이터를 나타내는 셀을 표현

	<th> / <td>는 <tr>태그의 자식요소로 사용



 - thead : tableheader, 테이블의 최상단행으로 고정
 - tfoot : tablefooter, 테이블의 초하단행으로 고정

 - tbody : table body thead 와 tfoot의 중간에 배치

 - caption : 테이블의 이름 ( 테이블 바깥상단에 위치)


+ <td> 또는 <th>의 셀 확장 속성

 - colspan : 열을 확장하는 속성, 셀이 차지할 칸 수를 지정

 - rowspan : 행을 확장하는 속성, 셀이 차지할 행 수를 지정
