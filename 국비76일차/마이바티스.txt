+ 장점
 - JDBC의 동작을 캡슐화되어 제공된다
 (Connection -> PreparedStatement -> 쿼리 수행 -> ResultSet -> close)

 - SQL코드와 프로그램코드를 분리하여 관리할 수 있다

 ** iBatis는 MyBatis의 예전 이름이다
	iBatis는 Apache재단에서 프로젝트로 존재하던 것
	MyBatis는 프로젝트가 구글코드로 넘어가면서 이름을 바꾼것

	iBatis - jdk 1.4 이전
	MyBatis - jdk 1.5 이후

+ 마이바티스 설정
 - 필요 라이브러리 설치
 	mtbatis-x.x.x.jar
	ojdbcX.jar

 - 마이바티스 설정파일 작성 ( XML 파일)
	configuration 파일을 작성한다.
	XML파일 형태로 작성한다
	마이바티스 접속 설정을 적는 파일이다
	마이바티스 유저가이드에서 복사하여 수정해서 사용한다
