※ Wrapper 클래스
	기본 데이터타입을 포장하고 있는 클래스
	기본 데이터 타입의 데이터를 객체로 만들어서 사용할 수 있게 해주는 클래스
	기본형 데이터를 참조형 변수로 사용할 수 있게 해준다.

	기본형		Wrapper
	booela		Boolean

	char		Character
	
	byte		Byte	

	short		Short
	
	int		Integer

	float		Float

	double		Double	


* Wrapper 클래스를 사용하는 이유
	기본형은 자바클래스들의 상속구조에 참여하지 못한다.
	기본형들은 Object클래스의 자식이 아니다.
	
+ Wrapper 클래스를 이용하여 참조형으로 표현할 수 있게 된다.
	기본형데이터들을 참조형으로 사용해야할 때 사용된다.
