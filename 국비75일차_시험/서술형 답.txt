Page : 하나의 jsp를 처리하는 동안만 유지되는 영역

Request : 하나의 요청을 처리하는 동안만 유지되는 영역

Session : 한 브라우저에서 작업하는 동안 유지되는 영역 세션 시간이 지나거나
브라우저를 닫으면 끊김 

Application : 어플리케이션이 실행되는 동안 유지되는 영역 서버가 켜지는 동안 계속 유지


--------------------------------------------------------------------------
자바스크립트를 이용해서 비동기 형식으로 http 통신을 하는 기술이다 현재 보여지는 화면의 새로고침 없이 현재 웹페이지 특정영역에 db 조회 정보를 보여주거나, 새로운 view를 제시할 수 있음
-------------------------------------------------------------------------


request 객체 : request.setAttribute("파라미터 이름", view에 넘길 파라미터); 형식으로 view에 전달한다.

session 객체 : session.setAttribute("파라미터 이름", view에 넘길 파라미터); 
형식으로 view에 전달한다.
