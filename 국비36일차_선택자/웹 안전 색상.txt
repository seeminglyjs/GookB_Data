 + 웹 안전 생산
	- RGB 기준으로 표현 가능한 색상의 조합이 16,777,216 가지 이다
	
	R : 0 ~ 255, 256가지(2^8)
	G : 0 ~ 255, 256가지(2^8)
	B : 0 ~ 255, 256가지(2^8)

	RGB로 조합가능한 색상표현 : 256 * 256 * 256 ( 2^24)

- RGB표현의 모든 색상을 구분해서 웹에서 나타내기에는 현실적으로 어려움

- 색상을 간소화하여 웹 표준 안전 색상을 정한 것

- 운연체제, 브라우저, 장치 등에 구애받지 않고 표현 가능한 색상들을 정함

- 0 ~ 255 수치를 00,33,66,99,cc,ff만 사용해서 표현한다.
- 간소화하면 0,3,6,9,c,f 로 사용할 수 있다.

- 256 가지를 6가지로 줄인 것
 -> 6 * 6 * 6 = 216가지 생삭이 된다.
