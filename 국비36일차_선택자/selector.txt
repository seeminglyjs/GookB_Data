 + 선택자, Selector
 - html 문서에서 특정 요소를 선택하는 css 문법
 - 선택된 요소들에게 일관적으로 스타일을 적용할 수 있다

+ 선택자의 종류

 - 전체 선택자
	* { }

 - 태그 선택자
	tagName {}

 - 아이디 선택자
	#idName{}	

 - 클래스 선택자
	.className{}

 - 복합 선택자
	둘 이상의 선택자를 복합적으로 사용하는 방법
	선택자에 포함된 요소들의 관계를 따져서 태그들을 선택한다.
	
	1. 하위(자손) 선택자 - Descendent
		E F
		E 요소의 하위 요소인 F들을 선택	
		EX) div p
		div 태그의 자손들 중에 모든 p 태그를 선택
		#board .title
		id가 board인 요소의 자손들 중에 클래스가 title인 요소
	
	2. 자식 선택자 - Child
		E > F
		E 요소의 자식 요소인 F들을 선택

		
	3. 형제 선택자 - Sibling
		E + F
		E의 인접한 형제 F
		(E요소의 바로 뒤에 오는 F요소 한 개)

		E ~ F
		E의 일반적인 형제 F
		(E요소를 뒤따르는 모든 F요소 전부)



 - 속성 선택자	
	요소의 속성을 이용하여 선택한다.
	
	[attr] attr속성을 가지고 있는 태그
	
	[attr="val"] attr 속성의 값이 val값과 같은 태그
	(속성의 값이 완전 일치해야함, 띄어쓰기까지 전부 같아야함)
	
	[attr~="val"] attr속성의 값으로 val단어를 완전포함하는 태그
	(띄어쓰기로 구분되어 있는 단어가 같은 값을 가지고 있으면 선택)
	
	[attr^="val"] -> attr속성의 값이 val로 시작하는 태그
	[attr$="val"] -> attr속성의 값이 val로 끝나는 태그
	[attr*="val"] -> attr속성의 값이 val을 부분 포함하는 태그
	[attr|="val"] -> attr속성의 값이 "val"과 같거나 "val-"로 시작하는 태그

 + 태그의 id, class 속성
 - 둘 모두 global attribute, 공용 속성이다
 - 모든 태그들이 다 사용할 수 있는 속성

 - id 속성
	html 문서에서 요소들을 서로 구분하기 위해서 사용
	문서 내에서 고유한 값으로 부여한다.
	Javascript Event 처리코드르 적용할 때 많이 사용한다.
	CSS를 적용할 때에도 많이 사용한다.

	#기호와 연계해서 사용한다.
		

 - class 속성
	태그들을 같은 그룹으로 편성할 때 사용한다.
	여러 태그들에 같은 클래스값을 적용하여 같은 그룹으로 표현한다
	같은 그룹으로 표현된 태그들에게 같은 스타일을 적용할 수 있다
	(자바스트립트 코드도 적용할 수 있다)
	
	. 기호를 이용하여 표현한다
	
	클래스값은 여러 개의 값을 부여할 수 있다
	각각의 값은 띄어쓰기로 구분한다
	ex) <p class="colorRed blue menu">테스트</p>
