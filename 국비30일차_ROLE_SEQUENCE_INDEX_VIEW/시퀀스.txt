+ 시퀀스, Sequence
 - 연속된 숫자를 생성하는 객체

 - 정수값으로 생성한다

 - 테이블에서 사용할 유일한 숫자를 자동으로 생성할 때 사용한다.
  -> PRIMARY KEY로 사용할 값을 생성한다


 - 테이블마다 각각 시퀀스르 따로 생성해서 전용으로 사용한다.


+ 시퀀스 생성 구문
	CREATE SEQUENCE 시퀀스명;
	-> 1부터 1씩 증가하는 시퀀스를 생성한다.

	CREATE SEQUENCE 시퀀스명
	[START WITH N]		-> 시작값
	[INCREMENT BY N ]	-> 증가값
	[MAXVALUE N | NOMAXVALE]-> 최대값
	[MINVALUE N | NOMINVALUE]-> 최소값
	[CYCLE | NOCYCLE]	-> 순환 여부
	[CACHE | NO CACHE]	->미리 생성할 숫자의 개수

+ 시퀀스 변경 구문

	ALTER SEQUENCE 시퀀스명
	[INCREMENT BY N ]	-> 증가값
	[MAXVALUE N | NOMAXVALE]-> 최대값
	[MINVALUE N | NOMINVALUE]-> 최소값
	[CYCLE | NOCYCLE]	-> 순환 여부
	[CACHE | NO CACHE]	->미리 생성할 숫자의 개수

** CREATE SEQUENCE 시퀀스명 빼고는 모두 사용가능


+ 시퀀스 사용 방법
	시퀀스명.CURRVAL - 현재 시퀀스 값
	시퀀스명.NEXTVAL - 다음 시퀀스 값
	**CURRVAL은 NEXTVAL을 한번이라도 사용한 후에 사용할 수 있다.


+ 시퀀스 삭제
	DROP SEQUENCE 시퀀스명

+ 시퀀스 자료사전
	USER_SEQUENCES
